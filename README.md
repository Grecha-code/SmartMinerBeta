<h1>1. Профиль работника</h1>
<h3>В профиле работник может 
изменить параметры 
приложения, настроить свой
профиль, а также при статусе
“Админ”(начальник шахты) может
редактировать тестирование или же
дать рекомендации 
конкретному работнику</h3>

<h1>2. Тестирование работника</h1>
<h3>Перед входом на рабочую зону
сотрудник обязан пройти небольшое
тестирование, в ходе которого 
проверяются знания работника, на
тему техники безопасности. Начальник
может менять это тестирование, так что
оно не будет всегда однотипным</h3>

<h1>3. Статистика</h1>
<h3>Самая интересная вкладка. Здесь
можно посмотреть  данные с
устройства, статистику своих коллег.
И самое главное, система бонусов за
соблюдение техники безопасности</h3>
